# ChatGPT Print Message - Install Instructions

1. Download this extension folder and unzip it (or clone it) to a known location on your computer.
2. Open Google Chrome and navigate to chrome://extensions/.
3. In the top-right corner, enable Developer mode.
4. Click Load unpacked.
5. Select the folder that contains manifest.json.
6. Open chatgpt.com, click the message menu on any message, and choose Print this message.

Print color mode:

- Click the extension button in Chrome to choose the default Light or Dark print mode.
- Click the extension button in Chrome to turn the extension On or Off.
- Refresh the ChatGPT page after turning the extension On or Off.
- In ChatGPT's message menu, choose Print this message to print with the selected mode.

Note: No background.js is required for this Manifest V3 extension; all logic lives in the content script and popup.
